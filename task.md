Задание

Этап 2. Архитектура

Предложите 2 варианта архитектуры системы. Для каждого варианта укажите:

    плюсы
    минусы
    какой вариант лучше и почему

Что сдавать

Файл группа_фамилия_имя.md в репозитории проекта.

    Откройте репозиторий.
    Нажмите Add file.
    Выберите Create new file.
    Файл задания должен называться task.md.
    Создайте отдельный файл с ответом: группа_фамилия_имя.md.
    Вставьте текст работы в файл группа_фамилия_имя.md.
    Нажмите Commit changes.

Task

Task

Stage 2. Architecture

Propose 2 architecture options for the system. For each option include:

    system diagram

    disadvantages
    which option is better and why

Submission

Upload a group_surname_name.md file to the project repository.
How to upload to GitHub

    Open your repository.
    Click Add file.
    Select Create new file.
    The assignment file is named task.md.
    Create your answer file: group_surname_name.md.
    Paste your work into group_surname_name.md.
    Click Commit changes.
